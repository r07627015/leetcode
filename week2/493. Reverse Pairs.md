
## Python
```python
class Solution:
    def reversePairs(self, nums):
        def merge_sort(left, right):
            # 用 merge sort：
            # 左右兩半排好序後，可以快速統計跨區間的 reverse pairs
            if left >= right:
                return 0

            mid = (left + right) // 2

            # 左半答案 + 右半答案
            count = merge_sort(left, mid) + merge_sort(mid + 1, right)

            # 統計跨左右兩半的答案
            j = mid + 1
            for i in range(left, mid + 1):
                while j <= right and nums[i] > 2 * nums[j]:
                    j += 1
                count += j - (mid + 1)

            # merge 兩個已排序區間
            temp = []
            i, j = left, mid + 1

            while i <= mid and j <= right:
                if nums[i] <= nums[j]:
                    temp.append(nums[i])
                    i += 1
                else:
                    temp.append(nums[j])
                    j += 1

            while i <= mid:
                temp.append(nums[i])
                i += 1

            while j <= right:
                temp.append(nums[j])
                j += 1

            nums[left:right + 1] = temp
            return count

        return merge_sort(0, len(nums) - 1)
```

- Time Complexity: `O(n log n)`
- Space Complexity: `O(n)`
