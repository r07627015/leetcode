
## Python
```python
class Solution:
    def reversePairs(self, nums):
        def merge_sort(left, right):
            # 基底情況：如果區間只有一個元素或為空，無法構成 pair，回傳 0
            if left >= right:
                return 0

            # 1. 分治法 (Divide)：將陣列對半切分
            mid = (left + right) // 2

            # 遞迴計算左半邊與右半邊各自內部的 reverse pairs 數量
            # 執行完這兩行後，左半區間 nums[left:mid+1] 和右半區間 nums[mid+1:right+1] 會分別被排序好
            count = merge_sort(left, mid) + merge_sort(mid + 1, right)

            # 2. 統計跨區間的 Reverse Pairs (Conquer)：
            # 此時左半部與右半部已各自「由小到大」排好序
            # 題目要求 nums[i] > 2 * nums[j]，其中 i 在左半部，j 在右半部
            j = mid + 1
            for i in range(left, mid + 1):
                # 利用單調性 (Monotonicity)：
                # 如果 nums[i] > 2 * nums[j]，那麼 nums[i+1] (因為更大) 
                # 一定也大於目前的 2 * nums[j]，所以指標 j 不需要重頭開始。
                while j <= right and nums[i] > 2 * nums[j]:
                    j += 1
                
                # 符合條件的個數就是 (j 目前的位置) 減去 (右半部的起始位置)
                count += j - (mid + 1)

            # 3. 合併 (Merge)：標準的歸併排序合併步驟
            # 目標是將兩個已排序的區間合併成一個大的排序區間，供上一層遞迴使用
            temp = []
            i, j = left, mid + 1

            while i <= mid and j <= right:
                if nums[i] <= nums[j]:
                    temp.append(nums[i])
                    i += 1
                else:
                    temp.append(nums[j])
                    j += 1

            # 處理左半部或右半部剩餘的元素
            while i <= mid:
                temp.append(nums[i])
                i += 1
            while j <= right:
                temp.append(nums[j])
                j += 1

            # 將排序好的結果寫回原陣列 nums，確保下一層遞迴處理的是排序好的資料
            nums[left:right + 1] = temp
            return count

        # 啟動歸併排序，範圍從索引 0 到 n-1
        return merge_sort(0, len(nums) - 1)
```

- Time Complexity: `O(n log n)`
- Space Complexity: `O(n)`
