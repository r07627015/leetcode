
## Python
```python
class Solution:
    def wiggleSort(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """

        nums.sort()  # 由小到大排序
        
        n = len(nums)

        mid = (n + 1) // 2  # 這是取中間，適用於n為奇數偶數的通用，無條件捨去小數後
        # 如果是奇數長度 n，偶數索引本來就多一格，所以中位數要包含在偶數索引列
        # 以下為根據索引0開始來調整
        left = mid - 1
        right = n - 1

        result = [None] * n  # 最後不能直接回傳，要放回nums

        # 偶數索引放前半，從尾端開始拿，n 是不包含的終點
        for i in range(0, n, 2):
            result[i] = nums[left]
            left = left - 1

        # 奇數索引放後半，從尾端開始拿，n 是不包含的終點
        for j in range(1, n, 2):
            result[j] = nums[right]
            right = right - 1

        # 複製回原陣列
        for i in range(n):
            nums[i] = result[i]
```

- Time Complexity: `O(n log n)`
- Space Complexity: `O(n)`
